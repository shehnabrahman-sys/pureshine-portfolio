Problem Solving Skills

How I find solutions to challenges at PureShine Auto Detail

Why Problem Solving?

Problem solving is a critical skill for any employee at PureShine Auto Detail. In a mobile car detailing business unexpected challenges can arise at any time such as equipment failure, difficult stains on a vehicle, a customer changing their booking at the last minute or bad weather affecting the schedule. Being able to think quickly and find effective solutions to these challenges is what separates a good employee from a great one.

I have developed strong problem solving skills through my IT studies and web development projects.

How I Solve Problems

Here is my approach to solving problems at work and in my studies:

Identify: Clearly understanding what the problem is before jumping to a solution
Research: Looking up information and resources to help find the best solution
Plan: Thinking through different options and choosing the most effective one
Execute: Putting the solution into action carefully and efficiently
Review: Checking that the solution worked and learning from the experience

Examples of My Problem Solving

Here are real examples of problems I have solved during my studies and web development projects:

  • Fixed a broken CSS layout by researching flexbox and applying the correct properties
  • Solved a file path error that was preventing the CSS file from loading on the website
  • Fixed HTML validation errors by carefully reading the W3C error messages and correcting the code
  • Resolved a JavaScript form validation issue by debugging the code step by step
  • Found a solution to display the website correctly on different screen sizes using CSS media queries
  • Helped a classmate fix their Notepad file that was saving as .txt instead of .html
